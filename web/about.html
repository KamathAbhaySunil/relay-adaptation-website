Understanding the Adaptive Protection Engine

How it Works

RelayAdapt is an end-to-end framework for microgrid protection. It solves the critical problem of Relay Blindness caused by modern Inverter-Based Resources (IBRs).

Core Architecture

  • System Model: A 4-bus nodal analysis engine built with NumPy.
  • Adaptive Logic: Real-time recalculation of \(I_s\) (Pickup) and \(TMS\) (Time Multiplier) settings.
  • Machine Learning: A Multi-Layer Perceptron (MLP) that predicts optimal settings in milliseconds.

The Visualization Engine

The dashboard you see on the main page uses Chart.js with a logarithmic scale to replicate standard industrial protection coordination studies. It allows engineers to stress-test settings by moving sliders and observing the "Adaptive Shift" in the Relay curves.
